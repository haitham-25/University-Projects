# 🧠 ResNet50 Image Classification Project

Deep Learning project that uses **Transfer Learning with ResNet50** to
build an image classification model.

## 📌 Project Overview

This project demonstrates how to use a **pre-trained CNN (ResNet50)** to
solve an image classification problem efficiently.

## 🎯 Objectives

-   Apply Transfer Learning
-   Build a high-accuracy image classifier
-   Reduce training time and overfitting
-   Understand deep learning workflow end-to-end

## 🏗️ Project Pipeline

1.  Import Libraries (TensorFlow, Keras, NumPy, Matplotlib,
    Scikit-learn)
2.  Data Preparation & Augmentation
3.  Transfer Learning using ResNet50
4.  Model Compilation (Adam + categorical_crossentropy)
5.  Model Training
6.  Evaluation & Predictions
